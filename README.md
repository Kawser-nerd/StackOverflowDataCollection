# XLibRec
XLibRec is a tool which can recommend similar libraries across different programming languages. This tool helps developers in developing functionally similar software applications across different programming languages as well as library migration among software applications developed in different programming languages. User of this tool needs to give input the source library for which they will get the analogical libraries in their target programming languages.

This tool is developed based on the discussion of a library in Stack Overflow and library description available in Libraries.IO
